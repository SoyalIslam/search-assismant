Core Analysis & Clustering Patterns

The Patterns Behind the Data

Analyzing 100 integrations reveals that the SaaS developer ecosystem is highly structured, with OAuth2 and API keys dominating authorization models.

  • Auth Dominance: OAuth2 (56% of apps) is the standard for user-facing applications (CRMs, Productivity, Comms), while API Keys / Tokens (43%) dominate developer infrastructure and data platforms.
  • Self-Serve Openness: Developer sign-up is open and self-serve for 79% of studied apps. The most developer-friendly categories are Productivity, Developer Tools, and Communications (100% self-serve sandbox setup).
  • Gated Barriers: 20% of apps cannot be built as agent toolkits today. The primary blocker is the absence of a public developer API or a complete partner gate requiring sales contact (e.g., DealCloud, Otter AI, iPayX).
  • Easy Wins vs. Outreach: Easy wins include modern SaaS apps with robust REST/GraphQL APIs and self-serve sandboxes (e.g., Twenty, Attio, Plain). High blockers require manual business development or developer partnerships.

Research Agent Architecture & Verification Loops

Automated Research Pipeline

To scale API research, we designed a Python-based research pipeline using DuckDuckGo search queries and Gemini 1.5 Flash to synthesize unstructured web results.

1

Search Queries

The agent generates automated queries targeting the developer docs, authentication requirements, and developer tier structures.

2

Gemini synthesis

A structured JSON schema prompt instructs the model to extract and validate details, saving intermediate outputs to avoid rate limits.

3

Human-in-the-loop corrections

A verification script matches a sample against hand-verified developer docs to ensure accuracy.

Developer Repository & Execution Guide

How to Run the Research Agent Pipeline

The repository is managed via uv to facilitate fast dependencies setup and isolated Python environments. Follow the commands below to execute the pipeline:

# 1. Clone the repository and navigate into the folder
git clone <repo-url> && cd composis

# 2. Set your Gemini API key in your terminal
export GEMINI_API_KEY="your-gemini-api-key"

# 3. Run the research agent pipeline using uv (downloads dependencies automatically)
uv run --with google-generativeai --with duckduckgo-search --with requests src/research_agent.py

# 4. Verify accuracy metrics against ground-truth sample
python3 src/verify_accuracy.py

# 5. Re-run analysis & generate assets for the dashboard
python3 src/analyze_patterns.py && python3 src/generate_dashboard_assets.py
